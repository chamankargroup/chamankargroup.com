# Chamankar Group Website

Official website for Chamankar Group - A trusted name in the construction industry for over 25 years.

## Features

- Responsive design for all devices
- Project showcase with image galleries
- Admin panel for managing projects
- Contact form
- About us section

## Tech Stack

- Vue.js 3
- Vue Router
- Tailwind CSS
- LocalStorage for data persistence

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Admin Access

Admin panel available at `/admin` route.
